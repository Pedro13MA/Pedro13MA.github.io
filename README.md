# Spotter Intelligence Hub — Landing Page

Static marketing site for [Spotter Intelligence Hub](https://github.com/Pedro13MA/spotter-intelligence-hub).  
Completely independent from the Python backend — no dependencies, no build step.

## Files

```
spotter-landing/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Local preview

Open `index.html` in a browser, or serve locally:

```bash
python -m http.server 8080
# http://localhost:8080
```

## GitHub Pages

1. Create a new repo (e.g. `spotter-landing` or use `username.github.io`)
2. Push these files to the `main` branch
3. **Settings → Pages → Source:** Deploy from branch `main` / root
4. Site live at `https://<user>.github.io/<repo>/`

## Customize

- Telegram link: `https://t.me/spotter_dealss` in `index.html` (CTA section)
- Footer year / version as needed
